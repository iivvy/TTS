# Custom TTS Demo
An advanced Text-to-Speech (TTS) application with voice cloning, multilingual support, and real-time audio visualization. Generate high-quality speech from predefined voices or record and clone new voices directly from your browser.
## Installation
#### clone this repository
```
git clone https://github.com/iivvy/TTS.git
```

```
cd TTS
```
### Install dependencies
```
pip install -r requirements.txt
```
Make sure your NumPy version is compatible with TTS, SciPy, and Torch (recommend numpy<2)
## Usage
Run the main application
```
streamlit run app.py
```
To test live voice recording and cloning run :
```

streamlit run voice_clone.py
```
